#Ejecutar en desarrollo 

1. Clonar repositorio
2. Ejecutar el comando de :

```
yarn install
```

3. Tener instalado Nest CLI

```
npm i -g @nestjs/cli
```

4. Lebamntar la base de datos

```
docker-compose up -d
```


# Stack usado
* Mongo DB
* Nestjs
